# Apex Exotics – Full Stack Portfolio Project

Apex Exotics is a responsive, production-ready exotic car rental web application built to demonstrate full stack development skills, UI/UX implementation, and real-world system design. The project delivers a complete vehicle booking platform — from customer browsing and reservations through to owner car management — with a clean, polished interface and smooth animations throughout.

Developed using React, Node.js, and MongoDB, the application features a role-based system where users can upgrade to an owner role, unlocking a dashboard to list and manage their vehicles. It showcases the ability to build scalable, maintainable full stack systems while delivering a seamless experience across all devices.

🚀 **Live Demo:** *https://apex-exotics.vercel.app/*

## Highlights

- Fully responsive, mobile-first design with tailored layouts for all screen sizes
- Role-based authentication with JWT — users can upgrade to an owner role to access the owner dashboard
- Owner dashboard for listing, managing, and tracking car availability
- Dynamic car search with location, pickup date, and return date filtering
- Image uploads handled via ImageKit with cloud storage
- MongoDB Atlas collections for users, cars, and bookings
- Smooth page transitions and animations powered by Framer Motion
- Clean and modular component architecture with shared app context
- Production deployment with Vercel
